<h1>Encriptador de texto</h1>

-Estado del proyecto: Finalizado.

Para ejecutar el encriptador, debes poner:

````npm install react````

-Encriptador de texto para verificar si aun tengo la oportunidad
de seguir con la becha de Alura.
Se ve de esta manera el encriptador de manera web.

![Imagen de como se ve el proyecto]<img width="919" alt="Captura de pantalla 2024-08-13 221740" src="https://github.com/user-attachments/assets/34040a97-f7d6-43f3-a5f9-dd31eef43784">

Funcionamiento:
-En el cuadro de ingrese le texto aqui se coloca la palabra que se quiere encriptar.
Se presiona el botón encriptar y el resultado encriptado aparece la lado derecho.
Se agrega el botón de copiar debajo del resultado, para copiar el texto encriptado.
Se puede presionar el botón y se copia el texto para agregrarlo al cuadro izquiero,
para desencriptarlo.
Agregue el botón de limpiar para tener los cuadros limpios.
